一个基于C++（SFML）开发的自走棋策略游戏，具有完整的游戏机制和闯关对战功能。
🎮 游戏功能
购买棋子、升级棋子、放置棋子、自动对战、闯关升级……
📦 安装和运行
方式一（目前仅支持该方式）：Visual Studio 开发
使用 Visual Studio 2022 打开 AutoChess.sln
配置SFML,libtorch依赖库，然后按 F5 开始调试运行
